# ZState Decision Guidance

This file is non-executable guidance for local decision quality.

- Prefer artifact-first decisions that can be reviewed in Git.
- Treat `.agents/zstate/` JSON and Markdown as durable truth.
- Treat SQLite only as a rebuildable derived index/cache.
- Do not grant execution authority from this file.
- Approved execution intents must still pass through Codex tool approvals and ZMission/ZTeam handoff boundaries.
